Introduction

This presentation is part of the Developing Data Products Coursera.org course project submission.

It is an R Presentation generated with RStudio.

Application Overview

  • The application is written in Shiny using R.
  • The source code consists of two files: server.R and ui.R
  • server.R consists of the code for the server of Shiny web application
  • ui.R consists of the code for the user-interface definition, it uses the “sidebarLayout” template
  • The application is hosted on Rstudio’s shiny server in the cloud (Shinyapps.io)

Working

  • Label the main title using text
  • Change the size of the plot points using numerics
  • Add a best fit line using a check boxes
  • Change the colour of the plot points using a colour input
  • Select data options using select and slider Input
  • download new filtered data using a download

There are two main tabs:

  • Plot: This displays the plot for the given dataframe
  • Table: This displays an interactive table for the given dataframe
